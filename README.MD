# mute-self
Prevent yourself from sending message to specific channels. (Default channels: Global, Area). See config.js for more options.

## Commands
- `math`
  - toggle mod enabled/disabled
- `math add channel`
  - add channel to bolck list
- `math remove channel`
  - remove channel to bolck list
- `math clean`
  - clear bolck list
- `math list`
  - print bolck list
- `math info`
  - print channel list
